# AI Resume Screening System

An NLP-based resume screening system that predicts job roles using machine learning and Streamlit.

## 🚀 Features
- Resume parsing
- Job role prediction using TF-IDF
- Skill extraction
- Streamlit web app

## 🛠 Tech Stack
- Python
- Scikit-learn
- NLP (TF-IDF)
- Streamlit

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run app.py
